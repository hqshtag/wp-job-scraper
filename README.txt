Requires WP Job Manager.

== Description ==

**WP Job Scraper** will help you load hundred of job-ads from _multiple APIs_, parse them the way you desire and add them to your job listings via the _WP API_.


== Installation ==

1. Upload `wp-job-scraper.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Head to the admin menu for setting up the plugin.
